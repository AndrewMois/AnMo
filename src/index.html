Projects

GameTro

GameTro is a project for retro game lovers with the ability to play retro games, compete with your score, read retro game news and more. The heart of the project is games written from scratch in Vanilla JS. GameTro also uses News API and custom JS Scripts with JQuery. The design was invented from scratch in Figma and realised with HTML and CSS.
